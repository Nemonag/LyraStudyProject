# LyraStudyProject

Unreal Engine Lyra Starter Game의 입력 → 어빌리티 실행 구조를 분석하고,
이를 기반으로 단순화하여 구현한 프로젝트입니다.

## 핵심 내용

- DataAsset 기반 입력 관리 (InputConfig)
- InputComponent를 통한 입력 바인딩
- Gameplay Ability System (GAS) 적용
- AbilitySystemComponent를 통한 어빌리티 실행

## 참고

본 프로젝트는 LyraStarterGame을 기반으로 구조를 학습하기 위해 작성되었으며,
일부 기능은 단순화되어 있습니다.
